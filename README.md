# The Badger Biodata Initiative 

We are a student organization at UW Madison interested in the cross between programming and 

We gather to share tools, strategies, victories, and woes. We help each other 
figure things out. And we do it in the spirit of the Wisconsin Idea: start on 
campus, stay open, propagate outward.

## What's here

This repo is the source for The Burrows website, built with 
[Quarto](https://quarto.org) and deployed to GitHub Pages. It includes:

- **Blog** — posts from BRUG members on tools, workflows, and lessons learned
- **Resources** — annotated examples and tutorials organized by topic 
  (visualization, wrangling, reproducibility, HPC, HTC)
- **Presentations** — slides and materials from BRUG meetups
- **Events** — upcoming and past meetups

## Contributing

Contributing to The Burrows is itself an exercise in the reproducible workflow 
BRUG promotes. If you have a resource, post, or presentation to share, open a 
pull request. See [CONTRIBUTING.md](CONTRIBUTING.md) for details.
