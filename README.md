# The Badger Biodata Initiative 

We are a student organization at UW Madison interested in biology and programming! We organize groups to create projects, discover tools, and learn bioinformatics. 

# The "Ecosystem"

Our club website is The Ecosystem, a place where we post meetings, events, projects, and club highlights. The intent to evolve this website overtime; we are welcome to any feedback or questions! 

Contact Jayden Merrick (jjmerrick [at] wisc.edu) for questions regarding the website or The Badger Biodata club.
